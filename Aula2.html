Introdução ao Python com PyCharm

Max Streicher Vallim

O que é Python?

  • Python é uma linguagem de programação de alto nível, interpretada e de propósito geral.
  • É conhecida por sua sintaxe clara e legível, o que facilita o aprendizado e a manutenção do código.

Por que usar Python?

  • Fácil de aprender e usar.
  • Ampla comunidade e suporte.
  • Bibliotecas e frameworks poderosos.
  • Versatilidade: usado em web development, data science, automação, etc.

O que é PyCharm?

  • PyCharm é um ambiente de desenvolvimento integrado (IDE) para Python.
  • Oferece ferramentas como depurador, controle de versão, e suporte a frameworks.
  • Disponível em versões gratuita (Community) e paga (Professional).

Instalando o PyCharm

  • Acesse o site oficial do PyCharm: https://www.jetbrains.com/pycharm/download/
  • Baixe a versão Community (gratuita) ou Professional (paga).
  • Siga as instruções de instalação para o seu sistema operacional.

Criando um projeto Python no PyCharm

  • Abra o PyCharm e selecione "Create New Project".
  • Escolha um local para o projeto e configure o interpretador Python.
  • Clique em "Create" para iniciar o projeto.

Escrevendo seu primeiro programa em Python

  • No PyCharm, crie um novo arquivo Python: clique com o botão direito na pasta do projeto e selecione "New" > "Python File".
  • Dê um nome ao arquivo, por exemplo, `hello_world.py`.
  • Escreva o seguinte código no arquivo: 
    print("Hello, World!")
  • Execute o programa clicando com o botão direito no arquivo e selecionando "Run 'hello_world'".
