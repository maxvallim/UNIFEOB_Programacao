Comandos Básicos do Python

Max Streicher Vallim

Introdução ao Python

  • Python é uma linguagem de programação de alto nível, interpretada e de fácil leitura.
  • É amplamente utilizada em diversas áreas como desenvolvimento web, ciência de dados, automação e muito mais.

Comando print()

  • O comando `print()` é usado para exibir mensagens ou resultados na tela.
  • Exemplo: 
    print("Olá, Mundo!")

Variáveis

  • Variáveis são usadas para armazenar valores que podem ser reutilizados no código.
  • Exemplo: 
    nome = "Alice"
idade = 25
print(nome, idade)

Estruturas de Controle

  • As estruturas de controle permitem controlar o fluxo de execução do programa.
  • Exemplo de `if`: 
    if idade >= 18:
    print("Maior de idade")
else:
    print("Menor de idade")

Laços de Repetição

  • Laços de repetição são usados para executar um bloco de código várias vezes.
  • Exemplo de `for`: 
    for i in range(5):
    print(i)

Funções

  • Funções são blocos de código reutilizáveis que realizam uma tarefa específica.
  • Exemplo: 
    def saudacao(nome):
    print("Olá,", nome)

saudacao("Alice")

Listas

  • Listas são coleções ordenadas de itens que podem ser de diferentes tipos.
  • Exemplo: 
    frutas = ["maçã", "banana", "cereja"]
print(frutas)

Dicionários

  • Dicionários são coleções de pares chave-valor.
  • Exemplo: 
    aluno = {"nome": "Alice", "idade": 25}
print(aluno)

Conclusão

  • Estes são alguns dos comandos básicos do Python que você pode usar para começar a programar.
  • Pratique escrevendo seus próprios códigos e explorando mais recursos da linguagem.
